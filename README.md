 Python & Power BI Data Analysis Study Project

This is a study project created as a part of the course 'Data Analyst' in John Bryce College.

## Project Overview
- **Python (Jupyter Notebook)**: data loading, cleaning, analysis, and visualization using Pandas and Matplotlib
- **Power BI Dashboard**: interactive dashboard summarizing the analysis results.
